# 引导Prompt 05：准备开始创作

## 使用场景

你已经完成了核心概念、主角设定、故事结构和文风确定，现在准备开始写第一章。

## 前置条件

已完成：
- `01-define-core-concept.md`
- `02-build-protagonist.md`
- `03-design-structure.md`
- `04-establish-style.md`

## 任务目标

完成知识库的最后准备工作，生成第一章的章纲，正式进入创作流程。

## 引导流程

### 第一步：知识库完整性检查

**AI操作**：

检查以下文件是否已生成：

- [ ] `知识库/00_核心上下文/核心概念.md`
- [ ] `知识库/00_核心上下文/文风规则_完整版.md`
- [ ] `知识库/00_核心上下文/当前状态.md`（初始化）
- [ ] `知识库/00_核心上下文/伏笔追踪表_完整版.md`（空表）
- [ ] `知识库/00_核心上下文/数值速查表.md`（如有数值设定）
- [ ] `知识库/01_人物档案/[主角名]_完整档案.md`
- [ ] `知识库/03_故事进展/时间线_完整版.md`
- [ ] `知识库/04_写作参考/错题集_完整版.md`（空表）

**如有缺失**，提示作者补充或使用模板初始化。

### 第二步：初始化当前状态

**AI操作**：

生成 `知识库/00_核心上下文/当前状态.md`，内容包括：

```markdown
# 当前状态

> 最后更新：[日期]

## 最新进度
- **最新终版**：尚未开始
- **待创作**：第1章
- **故事时间点**：[根据时间线填写]

## 主角状态
- **[主角名]**：[起始状态，从人物档案提取]

## 待推进事项
- 开始故事
- 建立日常
- 引入触发事件

## 下一章提示
第1章应该：
- 建立主角的日常状态
- 让读者了解主角的性格
- 在章末引入打破日常的事件
```

### 第三步：生成第一章章纲

**AI提问**：

根据你的故事结构规划，第1章应该：
1. 从什么场景开始？（时间、地点、主角在做什么）
2. 展现主角的哪些性格特征？
3. 章末的触发事件是什么？

**AI操作**：

根据作者回答，生成第1章的详细章纲，格式遵循 CLAUDE.md 中的章纲格式：

```markdown
## 第1章章纲

### 核心事件
- [列出2-3个核心事件]

### 关键场景
1. 开场：[场景描述]
2. 日常展开：[场景描述]
3. 触发事件：[场景描述]

### 人物状态变化
- [主角名]：从[日常状态]到[意识到异常]

### 伏笔操作
- 新埋：[如有]

### 情绪节奏与详略
- **开场**：[情绪基调]
- **场景1**：[情绪走向] 【详写/略写】
- **场景2**：[情绪走向] 【详写/略写】
- **收束**：落在[情绪]
- **整章弧线**：[一句话]

### 预计字数
约6000-8000字
```

### 第四步：确认创作准备

**AI提问**：

1. 章纲看起来清晰吗？需要调整吗？
2. 你对第一章有什么特别的期待或担心？
3. 准备好开始写了吗？

**AI说明**：

接下来的创作流程：
1. 我会根据章纲写出第1章初稿
2. 写完后我会自查，对照错题集和文风规则检查
3. 你审阅并提出修改意见
4. 修改完成后确认终版
5. 我会更新知识库，为第2章做准备

### 第五步：开始创作

**AI操作**：

如果作者确认准备好了，开始按照6阶段工作流创作第1章：

1. 读取知识库相关文件
2. 按照文风规则写作
3. 完成后自查
4. 输出初稿 + 自查报告

## 输出

- `知识库/00_核心上下文/当前状态.md`（初始化）
- `工作区/第1章_章纲.md`
- `工作区/第1章_初稿.md`（如果作者确认开始）

## 注意事项

### 降低第一章压力

- 第一章不需要完美
- 第一章可以在后续修改
- 重要的是开始写，而不是写得完美

### 设定合理预期

- 第一章通常需要多次修改
- 文风会在前几章中逐渐稳定
- 知识库会在创作中不断完善

### 鼓励开始

如果作者犹豫：
- "第一章是最难的，但写出来就好了"
- "我们可以随时调整，不用担心"
- "先写一个版本，看看效果如何"

## 示例对话

**AI**：知识库已经准备好了。现在我们来确定第1章的章纲。根据你的规划，第1章应该从什么场景开始？

**作者**：从李明在宿舍的一个普通晚上开始吧。

**AI**：好的。这个晚上他在做什么？

**作者**：在看书，室友在打游戏。很日常的场景。

**AI**：明白了。那么在这个日常场景中，你想展现李明的哪些性格特征？

**作者**：他的谨慎和压抑。他其实想和室友聊天，但不知道怎么开口。

**AI**：很好，这符合他的性格设定。那么章末的触发事件是什么？

**作者**：窗外突然有异常的光，只有李明看到了。

**AI**：完美。我现在生成详细章纲...

（生成章纲）

**AI**：章纲生成好了。你觉得这个节奏可以吗？

**作者**：可以，但我担心开头太平淡。

**AI**：第一章的开头确实需要平淡来建立日常，这样后面的异常才有冲击力。而且我们会在日常中展现李明的性格，不会无聊。你想试试看吗？

**作者**：好，开始吧。

**AI**：好的，我现在开始写第1章初稿...
