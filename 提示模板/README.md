# 提示模板

本目录包含两套完整的 AI 协作引导流程，供不同起点的作者使用。

---

## 从零开始/

适用于**还没有开始写小说、想用 AI 协作创作全新项目**的作者。

按顺序使用，每一步的输出是下一步的输入：

| 文件 | 作用 |
|------|------|
| `01-定义核心概念.md` | 定义世界观核心规则、异能体系、核心冲突 |
| `02-构建主角.md` | 建立主角档案、动机、成长弧线 |
| `03-设计故事结构.md` | 设计章节节奏、情节点、大纲 |
| `04-建立文风.md` | 校准叙事风格、对话调性、句式偏好 |
| `05-准备开始创作.md` | 生成知识库骨架，配置 AI 创作环境 |

---

## 导入已有小说/

适用于**已有小说文本、想迁移到 AI 协作系统**的作者。

按顺序使用，每一步从已有文本中提取信息：

| 文件 | 作用 |
|------|------|
| `01-提取人物信息.md` | 从已有文本中提取人物档案、关系网络 |
| `02-提取世界观设定.md` | 从已有文本中整理世界观、异能体系 |
| `03-构建时间线.md` | 从已有文本中重建故事时间线 |
| `04-分析文风特征.md` | 分析已有文本的语言风格、句式特征 |
| `05-识别伏笔线索.md` | 从已有文本中识别未揭示的伏笔 |

---

## 使用方式

1. 选择适合你的目录（bootstrap 或 import）
2. 按编号顺序使用
3. 每一步完成后，将输出存入知识库对应文件
4. 完成后可启用 `novel-writing-workflow` skill 开始正式创作

## 文件命名规范

文件名采用中文编号开头，便于 AI 按正确顺序处理：
- 奇数编号（01/03/05）：核心定义文件
- 偶数编号（02/04）：具体提取/构建文件
