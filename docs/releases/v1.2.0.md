# Craft Companion v1.2.0 发布说明

发布日期：2026-04-07

## 这次更新解决了什么

本版本重点解决“新用户上手慢、旧项目接入不顺、会话中断后难接力”的问题。

## 亮点

1. **统一 AI 入口**
- 新增 `AI入口_统一指令.md`
- 首轮统一分流，支持自然语言意图映射

2. **已有项目一键补齐**
- 新增命令：`craft-companion bootstrap-entry`
- 为旧项目补齐 `START_HERE.md`、统一入口、接力状态与基础骨架（不改正文）

3. **双层质量保障**
- 新增“执行层自查 + 评估层复核”方法
- 争议点采用“原文 + 建议改写块”并排保留

4. **连接错误排查**
- 新增 `docs/11-连接错误排查.md`
- 覆盖 `local error: tls: bad record MAC` 常见场景

5. **可运行验收**
- 新增命令：`craft-companion doctor`
- 一键检查入口文件、模板、接力格式和检查点目录

## 升级后建议执行

```bash
craft-companion bootstrap-entry
craft-companion doctor
```

## 升级前后对比

| 项目 | 旧版 | v1.2.0 |
|---|---|---|
| 首轮引导 | 靠人工提示 | 统一入口 + 固定分流模板 |
| 旧项目接入 | 手工补文件 | `bootstrap-entry` 一键补齐 |
| 质量检查 | 单层自查 | 双层复核（执行层+评估层） |
| 连接异常说明 | 缺失 | 提供排查与恢复建议 |
| 跑通验证 | 依赖人工判断 | `doctor` 一键体检 |

## 兼容性

- 无破坏性变更
- 旧项目可直接执行 `bootstrap-entry` 升级
