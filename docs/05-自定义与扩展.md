# 自定义与扩展

Craft Companion 是一个可扩展的本地创作框架。

## 你最可能扩展的地方

### 1. 提示模板
当前有两条主入口：
- `提示模板/从零开始/`
- `提示模板/导入已有小说/`

初始化完成后建议使用：
- `提示模板/通用流程/`

如果你要扩展，请优先遵守这两个入口的命名和分流逻辑，不要再新增旧式 `prompts/...` 路径。

### 2. 知识库结构
你可以在现有 5 层知识库基础上加自己的目录，但建议保持核心目录不变，避免工具失配。

### 3. CLI 工具
CLI 改动时，优先保证：
- 与当前模板路径一致
- 与 5 阶段工作流口径一致
- 不再使用 `NovelForge`、`template/` 等旧术语

### 4. 外部接入
如果接入 Cherry Studio、OpenClaw、柴油C6 等外部系统，建议统一首轮分流：
- 从零开始写新小说
- 导入已有小说

这样能显著降低用户迷路概率。

如果支持多 Agent，建议固定启用“执行层自查 + 评估层复核”的双层质量保障。
