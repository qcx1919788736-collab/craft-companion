## 🚀 快速开始

### AI助手
1. 阅读 [AI创作工作流指南.md](./AI创作工作流指南.md)
2. 按5阶段流程工作（章纲→选定→初稿→修订→终版）
3. 必读 `知识库/00_核心上下文/`，其他按需加载

### 人类用户
- 开始新章节：`"开始第XX章创作，给我3个版本的章纲"`
- 使用CLI工具：`node novel-cli.js help`

---

## 📁 目录结构

```
正文创作/
├── novel-cli.js              ← 自动化CLI工具
├── 知识库/                   ← 分层知识库（详见知识库/README.md）
├── 工作区/                   ← 章节创作文件
└── _归档/                    ← 历史文件归档
```

---

## 🛠️ CLI工具

```bash
node novel-cli.js new-chapter 21    # 创建第21章工作文件
node novel-cli.js archive 20        # 归档第20章
node novel-cli.js check             # 检查一致性
node novel-cli.js update-kb 20      # 显示知识库更新清单
```
