# 贡献指南

感谢你对 Craft Companion 的关注！欢迎任何形式的贡献。

## 如何贡献

### 报告问题
- 使用 GitHub Issues 提交 bug 报告或功能建议
- 描述清楚问题的复现步骤
- 附上相关错误信息或截图

### 提交代码
1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/xxx`
3. 提交修改：`git commit -m 'fix: xxx'`
4. 推送分支
5. 提交 Pull Request

## 贡献方向
- 提示模板优化
- CLI / MCP 工具改进
- 文档统一与引导优化
- 外部接入（Cherry Studio / OpenClaw / 柴油C6）

## 统一约定
- 项目名统一使用 **Craft Companion**
- 工作流统一使用 **5 阶段**
- 模板路径统一使用 **`提示模板/...`**
- 不再新增旧的 `NovelForge` / `prompts/...` / `template/...` 说法
