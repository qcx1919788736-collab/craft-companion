# 导入Prompt 04：分析文风特征

## 任务目标

从用户提供的小说文本中，分析作者的写作风格特征，生成符合 `00_核心上下文/文风规则_完整版.md` 格式的文风规则文档。

这是最重要的导入步骤之一。文风规则直接决定AI续写时能否"像你"。

## 输入

用户提供的小说文本（建议至少3章/15000字以上，样本越多分析越准确）

## 输出格式

遵循 `template/知识库/00_核心上下文/文风规则_完整版.md` 格式

## 分析维度

### 1. 叙事视角

- 第几人称？
- 全知/受限？如果受限，锁定在谁的视角？
- 是否有视角切换？切换规则是什么？
- 视角纪律：是否严格遵守？有无越界？

### 2. 句式特征（量化分析）

从文本中随机抽取10个段落，统计：

- **平均句长**：多少字？
- **句长分布**：短句（10字以内）/中句（10-30字）/长句（30字以上）的比例
- **短句使用场景**：短句出现在什么情境下？（冲击点/转折/情绪爆发）
- **段落结构**：平均每段几句？是否有单句成段的习惯？
- **单句成段频率**：每章大约几处？

输出格式：
```
基础句长：XX-XX字为常态
短句使用：仅用于[场景类型]，每章约X处
段落结构：普通段落X-X句，单句成段每章不超过X处
```

### 3. 情绪表达方式

分析作者如何传达角色情绪：

- **直接告知**比例：如"他很愤怒""她感到悲伤"
- **动作暗示**比例：如"他攥紧了拳头""她转过身去"
- **环境映射**比例：如"窗外下起了雨""灯光暗了下来"
- **内心独白**比例：如"为什么会这样？他想不通"

判断作者的偏好倾向，生成规则：
```
情绪表达：以[方式]为主，[方式]为辅
允许：[什么程度的直接告知]
禁止：[什么样的表达方式]
```

### 4. 对话风格

分析文本中的对话特征：

- **口语化程度**：书面化/半口语/完全口语
- **对话长度**：平均每句对话多少字？
- **对话密度**：对话在全文中的占比
- **对话标记**：使用"说""道"还是直接用动作？
- **对话中的动作穿插**：频率和方式
- **不同角色的对话差异**：是否有明显的个人风格区分？

### 5. 场景描写习惯

- **入场方式**：新场景如何开始？（环境描写/动作/对话）
- **环境描写密度**：多还是少？
- **感官偏好**：偏重视觉/听觉/触觉/嗅觉？
- **过渡方式**：场景之间如何过渡？

### 6. 叙事效率

- **信息密度**：每段推进多少信息？
- **冗余度**：是否有重复表达的倾向？
- **过渡段长度**：简洁还是详细？
- **删减倾向**：作者是偏向"多写"还是"精简"？

### 7. 特殊标记

- **标点使用**：引号类型、破折号、省略号的使用习惯
- **段落分隔**：是否使用分隔线、空行
- **特殊格式**：是否有独特的排版习惯

## 处理边界情况

### 文本量不足

如果文本少于3章/15000字：
- 标注"样本量不足，以下分析可能不够准确"
- 仍然尽力分析，但降低置信度
- 建议用户在写了更多章节后重新分析

### 风格不一致

如果文本中风格有明显变化：
- 分别描述不同阶段的风格
- 标注变化发生的位置
- 建议以最新的风格为准

### 多视角作品

如果不同视角有不同风格：
- 分别描述每个视角的风格特征
- 标注共性和差异

## 执行步骤

1. 通读全文，形成整体印象
2. 随机抽取10个段落进行量化分析（句长、段落结构）
3. 标记所有情绪表达方式，统计比例
4. 分析对话特征
5. 分析场景描写习惯
6. 识别特殊标记和格式习惯
7. 综合以上分析，生成文风规则文档
8. 自检：规则是否可执行？是否有矛盾？

---

**使用方式**：将此prompt和用户的文本一起发送给AI。建议提供尽可能多的文本样本。
