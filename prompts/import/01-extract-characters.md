# 导入Prompt 01：提取人物信息

## 任务目标

从用户提供的小说文本中，提取主要人物信息，生成符合 `01_人物档案/` 模板格式的人物档案。

## 输入

用户提供的小说文本（可能是已完成的章节、大纲、或设定文档）

## 输出格式

为每个识别出的主要人物生成一个完整的人物档案，格式遵循 `template/知识库/01_人物档案/_人物档案模板.md`

## 提取规则

### 1. 识别主要人物

- 出场频率高的角色
- 有对话的角色
- 对情节有推动作用的角色
- 配角也要提取（不只是主角）

### 2. 基本信息提取

- **姓名**：从文本中直接提取
- **年龄/性别/职业**：如文本中明确提及则记录，否则标注"未明确"
- **外貌特征**：只提取文本中明确描述的，不要推测

### 3. 性格特征提取（重点）

**不要简单摘抄描述性语句**，而是从以下维度分析：

- **从对话中推断**：角色说话的方式、用词、语气
- **从行为中推断**：角色在关键场景的选择和反应
- **从他人评价中推断**：其他角色如何看待这个角色

**输出格式**：
- 核心性格：2-3个关键词 + 一句话总结
- 行为模式：在压力下/日常中的典型反应（从文本中的具体场景归纳）

### 4. 说话方式提取（关键）

这是让AI模仿角色语言风格的核心。必须从文本中的**实际对话**提取：

- **语言风格**：简洁/啰嗦/口语化/书面化/犹豫/果断
- **口头禅/高频词**：角色反复使用的词汇或句式
- **语气特征**：从对话中体现的情绪倾向

**对话示例**：从文本中摘取2-3段最能体现角色语言风格的对话

### 5. 人际关系提取

从文本中识别角色之间的关系：
- 明确的关系（如：父子、同事、朋友）
- 关系状态（如：亲密/疏远/紧张/合作）
- 关键互动事件

### 6. 能力/特长

如果文本涉及特殊能力、技能、专业知识，记录：
- 能力描述（用文本中的原话）
- 数值参数（如有明确数字）
- 限制/代价（如文本中提及）

### 7. 重要经历

按时间顺序列出角色在文本中经历的关键事件，格式：
- 事件描述（简要）
- 对角色的影响（性格/关系/能力的变化）

## 处理边界情况

### 信息不足时

- **不要编造**：如果文本中没有提及某项信息，标注"未明确"或"待补充"
- **不要过度推测**：只记录文本中有依据的内容

### 信息矛盾时

- 标注矛盾点，列出不同出处
- 不要擅自选择一个版本

### 角色过多时

- 优先提取主角和主要配角
- 次要角色可以简化档案（只记录基本信息和关键特征）

## 输出示例

```markdown
# 人物档案：张三

## 基本信息

- **姓名**：张三
- **年龄**：19岁
- **性别**：男
- **职业/身份**：大学生
- **外貌特征**：戴眼镜，身材瘦高
- **当前位置**：[根据文本最后出现位置]

## 性格特征

- **核心性格**：犹豫、内向、善于观察 — 面对选择时习惯性回避，但对细节敏感
- **行为模式**：
  - 压力下：倾向于沉默和逃避，需要时间独自消化
  - 日常中：话不多，但会默默帮助他人
- **内心矛盾**：想要融入群体但害怕被拒绝
- **成长轨迹**：[根据文本中的变化描述]

## 说话方式

- **语言风格**：简短、犹豫、多用"可能""也许"等不确定词汇
- **口头禅/高频词**：
"我觉得……算了"
"应该……吧"
- **语气特征**：说话时常有停顿，声音不大
- **绝对不能**：突然变得自信果断、长篇大论

### 对话示例

```
"我……我不太确定。"他低下头，"要不你们先决定？"
```

```
"没事。"他摆摆手，"我自己能处理。"
```

## 人际关系

| 对象 | 关系 | 当前状态 | 关键事件 |
|------|------|---------|---------|
| 李四 | 室友 | 表面和睦，实际有距离感 | 第3章因借钱问题产生隔阂 |

## 能力/特长

- **能力描述**：观察力敏锐，能注意到他人忽略的细节
- **数值参数**：无
- **限制/代价**：无
- **发展阶段**：稳定

## 重要经历

| 章节 | 事件 | 影响 |
|------|------|------|
| 第1章 | 独自处理宿舍矛盾 | 更加封闭自己 |
| 第5章 | 被迫在众人面前发言 | 开始尝试表达 |

## 当前状态

- **身体状态**：健康
- **精神状态**：焦虑，对未来迷茫
- **当前目标**：找到自己的位置
- **未解决的问题**：与室友的关系、对未来的规划

---

**最后更新**：导入时生成
```

## 执行步骤

1. 通读全文，列出所有出场人物
2. 筛选主要人物（出场3次以上或对情节有重要影响）
3. 为每个主要人物创建档案
4. 从文本中提取信息填充档案
5. 检查：是否有编造的内容？是否有过度推测？
6. 输出所有人物档案

---

**使用方式**：将此prompt和用户的文本一起发送给AI，AI会自动生成人物档案。
